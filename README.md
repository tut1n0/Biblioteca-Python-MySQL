# Biblioteca Python MySQL

Sistema de gestión de biblioteca desarrollado en Python, con interfaz gráfica usando CustomTkinter y base de datos MySQL.

## Funcionalidades

- Inicio de sesión de empleados.
- Gestión de libros.
- Gestión de autores.
- Gestión de editoriales.
- Gestión de categorías.
- Gestión de socios.
- Gestión de préstamos.

## Tecnologías utilizadas

- Python
- CustomTkinter
- Tkinter
- MySQL
- PyMySQL
